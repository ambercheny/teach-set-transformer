Set Transformer Architecture

A compact map of the Set Transformer blocks: MAB, SAB, ISAB, PMA, and the point where an equivariant event representation becomes an invariant sample prediction.

One-Line Shape

Set Transformer follows the same broad pattern as Deep Sets, but replaces simple per-row encoding and fixed pooling with attention-based blocks.

input set X -> encoder(SAB or ISAB) -> event features Z -> decoder(PMA) -> sample prediction

Memory hook Encoder blocks keep one output per input event, so they are permutation equivariant. PMA aggregates across events, so the final sample-level output is permutation invariant.

Encoder: Equivariant Event Features

The encoder transforms the input event rows into event-level representations. Lee 2019 uses stacks of either SAB or ISAB blocks.

SAB Encoder

Encoder(X) = SAB(SAB(X))

Every event can attend to every other event. This can model interactions between cells, but full self-attention costs O(n^2).

ISAB Encoder

Encoder(X) = ISABm(ISABm(X))

Events interact through m learned inducing points. This reduces attention cost to O(nm) when m is smaller than n.

Both choices output one representation per input event. If the input rows are shuffled, the output rows shuffle the same way. That is permutation equivariance.

Decoder: Invariant Sample Output

After the encoder, the model has a set of event features Z. PMA uses learned seed vectors as queries to aggregate information from that set.

PMAk(Z) = MAB(S, rFF(Z))

For a single sample-level prediction, use k = 1. The output is one learned summary vector for the whole sample. Since the same set of event features produces the same summary regardless of row order, PMA is the point where the architecture becomes permutation invariant.

How This Maps To RBC Prediction

Flow cytometry interpretation.
Model part RBC interpretation Why it fits set data
Input X Rows of retained cell/event measurements from one sample. Rows have no canonical order.
SAB or ISAB encoder Creates one context-aware representation per cell/event row. Event representations reorder with input rows.
PMA decoder Aggregates event representations into one sample representation. Sample representation should not change when rows are shuffled.
Final prediction head Maps the sample representation to the sample-level target, such as hematocrit. The target belongs to the whole sample, not one row.
